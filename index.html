Weather Analysis

About this Project

Data used in this analysis was gathered from the Weather API for a single date. Information was then collected into a dataframe for plotting and statistical analysis using Python, Pandas and MatPlotLib tools.

When reviewing the data collected, the expectation was that there would be signifcant correlations between latitude and humidity. Conversely, I expected there to be no signifcant relationship between latitude and cloudiness or wind speed. As you will see by clicking each individual plot, part of this hypothesis proved true while there was one piece that was unexpectedly disproven.
